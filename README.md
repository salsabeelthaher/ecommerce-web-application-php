# E-Commerce Web Application

This project is a simple e-commerce web application developed using PHP and MySQL.

The system allows users to browse products, add items to the shopping cart, and complete purchases using PayPal payment integration. It also includes an admin dashboard for managing products and store data.

## Features

- Product catalog connected to a MySQL database
- Add to cart functionality
- User login and registration
- PayPal payment integration
- Admin dashboard to manage store content
- Product categories and item management

## Admin Dashboard

The admin panel allows administrators to:

- View store statistics
- Manage product categories
- Add, edit, and delete products
- Monitor transactions
- View system data through tables and charts

## Technologies Used

- PHP
- HTML
- CSS
- JavaScript
- MySQL
- PayPal Integration

## Project Purpose

This project was developed as part of an academic course to demonstrate web system development and database integration for an online store.
